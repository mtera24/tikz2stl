# tikz2stl
tikzで書かれたtexファイルから、3Dプリンタ用にSTLファイルを生成するツールです。
* tex -> pdf -> svg -> svg -> stl と段階を踏んで変換します
* 必須ツールとしてlatex, openSCADを使います。
* ***windows専用です。すみません***
* 現在はバッチファイルとして記述しています
## このパッケージ以外に必要なもの
* 変換したいtexファイル(当然ですが)
* platexが使える環境(TeXLiveなど)
* openSCADのインストール

## 使い方
1. 当ツール群と変換したい元texファイルを、同じフォルダに入れてください。
2. そのフォルダをカレントディレクトリとして、コマンドプロンプトを起動してください。
3. ``` tikz2stl.bat <元ファイル>.tex ```と入力すると、あとは自動で変換します。
4. 変換したSTLファイル名は、```<元ファイル>.stl```となります。

    - dia_marker = 5;           // マーカー底面の直径
    - height_marker = 1;           // マーカーの高さ(半球つぶし後の高さ)
    - size_base = 150;  // 土台のサイズ
    - thick_base = 0.3; // 土台の厚み
    - offset_marker = 10;      // マーカの土台端からの距離 (1cm = 10mm)
    - scale_figure = 2; // とりこむfigureの拡大率
